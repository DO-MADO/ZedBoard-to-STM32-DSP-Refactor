<h1> 🧬 Zynq 기반 DSP 시스템을 STM32H723 Nucleo 보드로 이식 및 리팩토링한 실시간 신호처리 프로젝트</h1>
ZedBoard(Linux + Python FastAPI 기반)에서 구현된 실시간 DSP 파이프라인을

STM32H723ZG Nucleo 보드로 <b>포팅</b>하고, <b>RS-485 시리얼 통신</b> 기반으로 <b>PC ↔ PCB</b> 간<br>
<b>실시간 데이터 송수신</b> 구조를 <b>재설계</b>한 프로젝트입니다.


<br>

## 📑 프로젝트 요약



- 기간 : 2025.10.20 - 2025.10.31
- 인원 : 3명 (하드웨어 1 · 펌웨어/PM 1 · 소프트웨어/통신 통합 1)
- 설명 : <br> ZedBoard 환경에서 운영되던 AD4858 기반 실시간 DSP 시스템을 STM32 MCU(CubeMX + HAL 기반 Bare-metal) 환경으로 포팅하여<br>
  RS-485 양방향 통신과 실시간 데이터 파싱 로직을 구현했습니다. PC → PCB 방향의 파라미터 전송은 정상 송수신까지 검증 완료되었으며,<br>
  PCB 내부의 실시간 DSP 파라미터 반영 로직은 통신 구조와 분리된 형태로 설계·검증 중입니다.
  <br>또한 Python 기반 PC 모니터링 툴을 .exe 및 .bat 형태로 패키징하여, Python 미설치 환경에서도 독립 실행 가능한 배포용 프로그램으로 완성했습니다.
